# ⚙️ Engineering Manager Skills

> Claude skills designed for Engineering Managers — the people responsible for team health, performance, delivery, and growth.

Engineering Managers operate at the intersection of people and product. The skills in this folder help you handle the recurring, high-stakes written work that comes with the role — faster and with better results.

## 👤 Who This Is For

You'll get the most out of these skills if you:

- Manage a team of 4–15 engineers
- Run regular 1:1s, performance cycles, and sprint ceremonies
- Write a lot — reviews, updates, feedback, escalations
- Want to use Claude seriously, not just for one-off prompts

---

## 💡 How EM Skills Differ from TL Skills

Tech Lead skills focus on **code and technical decisions** — review feedback, architecture records, technical documentation.

Engineering Manager skills focus on **people and process** — performance, growth, alignment, communication upward and downward.

Some skills overlap (e.g., both TLs and EMs give feedback) but the framing, tone, and output are calibrated for each role.

If you're an EM who also serves as Tech Lead, check the [TL folder](../tech-lead/) too — many EMs use both.
