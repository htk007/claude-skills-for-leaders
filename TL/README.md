# 🔧 Tech Lead Skills

> Claude skills designed for Tech Leads — the people responsible for code quality, architectural decisions, technical direction, and engineering standards.

Tech Leads sit at the intersection of hands-on engineering and team leadership. The skills in this folder help you handle the high-effort written and analytical work that comes with the role — without losing hours to blank-page syndrome.

## 👤 Who This Is For

You'll get the most out of these skills if you:

- Are the technical anchor of a squad or stream
- Review code regularly and care about how feedback lands
- Write ADRs, tech specs, or RFCs (or know you should but don't)
- Lead incident response and own post-mortems
- Want Claude to do the first draft, not the last word

---

## 💡 How TL Skills Differ from EM Skills

Tech Lead skills focus on **code and technical decisions** — review feedback, architecture records, incident documentation, technical specifications.

Engineering Manager skills focus on **people and process** — performance reviews, 1:1s, OKRs, promotion cases.

If you're a TL who also manages people, check the [EM folder](../engineering-manager/) too — many TLs use both.
