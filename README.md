# 🧠 Claude Skills for Leaders

> Practical, ready-to-use Claude skills for Engineering Managers, Tech Leads, Product Managers, and HR Managers.

Most Claude skills are built for developers writing code.  
This repo is built for **leaders** — the people managing teams, running reviews, writing feedback, and making high-stakes decisions every day.

---

## Why This Exists

Writing a performance review from scratch takes hours.  
A code review comment can start a conflict if the tone is off.  
A job description written in 20 minutes attracts the wrong candidates.

Claude can help with all of this — but only if you know how to use it well.  
These skills do the heavy lifting so you don't have to.

## 🚀 How to Use a Skill

1. Open [Claude.ai](https://claude.ai) → **Settings → Skills**
2. Click **New Skill**
3. Copy the content of any `SKILL.md` file from this repo
4. Paste it into the skill editor and save
5. Start using it in any conversation

No setup. No API keys. No code. Just copy, paste, and go.

---

## 📐 Skill Format

Every skill in this repo follows the same structure:

- **What it does** — the problem it solves
- **How to use it** — exact input format to paste into Claude
- **Example input / output** — so you know what to expect
- **Variations** — how to tweak it for different situations
